لحماية كلمات المرور PHP
 `password_hash()` لتجزئة كلمة المرور (Password Hashing)
 `password_verify()` للتحقق من تطابق كلمة المرور مع التجزئة

الهدف منه حماية كلمات المرور وعدم تخزينها كنص عادي.
لتجربة الكود على الرابط التالى_______ http://localhost/assigment2/_______


 أدخل كلمة المرور في الحقل.
اضغط **Hash Password** لإنشاء التجزئة.
 أدخل نفس كلمة المرور واضغط **Verify Password** للتحقق من التطابق.
 إذا تطابقت كلمة المرور مع التجزئة ستظهر نتيجة **Match**، وإذا لم تطابق ستظهر **No Match**.

- الكود يستخدم **Sessions** لتخزين التجزئة مؤقتًا أثناء عملية التحقق.
